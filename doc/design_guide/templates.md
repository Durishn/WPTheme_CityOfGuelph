# Page Templates for Guelph.ca

## COG Default Page
The COG_default_page template is used for content driven pages within the lowest levels of the guelph.ca sitemap, this template is built with flexibility in mind for our authors. It contains few mandatory modules along with a number of optional modules that can be added on a page-by-page basis by authors.

**Mandatory Fields**
* Breadcrumb - WP
* Page Title - WP - Requires user input
* Subsite Navigation - Plugin
* Page Content - Requires user input (allows )


**Optional Fields**
* Recent News - Plugin - Requires user input
* Recent Events - Plugin - Requires user input
* Page Notice/Alert - ACF - Requires user input

**Meta Fields**
* Table of Contents - Plugin
* Location in Navigation
* Page Excerpt - Requires user input
* Page Feature Image - Requires user input
* Visibility of Feature Image - Plugin
* Page Expiration
* Page Visibility
